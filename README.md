# Contact Form Project

## Overview
A simple contact form with HTML, CSS, and JavaScript validation.

## Contributors
- Florence Ndemi
- George Wainaina

## Files
- `index.html` - Contact form structure
- `styles.css` - Styling and design
- `script.js` - Form validation

## Usage
1. Open `index.html` in a browser
2. Fill out the form (name, email, message)
3. Click "Send Message"
4. Form validates input and shows success message

## Features
- Name validation (letters only)
- Email format checking
- Required field validation
- Responsive design
